# CodePilot AI Workspace

AI-native developer productivity platform built for IBM BOB Hackathon.

## Features
- Repository Analyzer
- AI Debug Assistant
- Documentation Generator
- JWT Authentication
- Real-time Logs
- Developer Insights

## Tech Stack
Frontend:
- React
- TypeScript
- Vite
- Tailwind
- shadcn/ui

Backend:
- NestJS
- Prisma
- PostgreSQL
- JWT
